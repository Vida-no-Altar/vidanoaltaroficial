Arquitetura futura

  • Cloudflare Pages para site público e Studio.
  • Cloudflare Workers para API própria.
  • Cloudflare D1 para dados estruturados.
  • Cloudflare R2 para imagens e arquivos.
  • Cloudflare Access como camada extra opcional.
  • Login futuro por e-mail e senha, com segunda etapa obrigatória.

Migração do Admin

  • /studio/ é a rota oficial nova.
  • /admin/ vira legado temporário com aviso claro.
  • /admin/legacy.html preserva o editor técnico existente.
  • /admin/auditor.html aponta para /studio/auditor/.
